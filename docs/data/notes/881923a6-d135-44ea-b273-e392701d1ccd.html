Test

[ -e FILE ] - True if FILE exists [ -d FILE ] - True if FILE exists and is a directory. [ -f FILE ] - True if FILE exists and is a regular file. [ -z STRING ] - True if the length if "STRING" is zero. [ -n STRING ] or [ STRING ] - True if the length of "STRING" is non-zero. 

http_status=$(curl ___)
if [ $http_status != "200" ]
    then
        echo "command failed"
fi

Test based on exit status of previously executed command

  • The ? variable holds the exit status of the previously executed command (the most recently completed foreground process).
if [ $? -eq 0 ]
then echo 'That was a good job!'
fi
