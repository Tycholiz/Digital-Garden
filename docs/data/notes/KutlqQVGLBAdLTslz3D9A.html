Cloud Development Kit

The AWS CDK is only for creating/deleting/managing infrastructure.

  • It spits out CloudFormation templates, which are what is used to build your AWS infra.

Essentially, if you are a developing an application, you will use the SDK of your programming language. If you are creating infrastructure, you will use the CDK.

vs SDK

You'd use the CDK to create a DynamoDB table. You'd use the SDK to read and write data in that table.
