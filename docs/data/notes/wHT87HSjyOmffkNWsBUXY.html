For

Run loop indefinitely, and stop when a certain condition is met

for (let turn = 0;; turn++) {
  if (state.parcels.length == 0) {
    console.log(`Done in ${turn} turns`);
    break;
  }
  // do your stuff here
}

for..of vs for..in

for..of should be used with arrays, and other iterable objects, like strings

  • an object is considered iterable if it adheres to the iterator protocols for..in should be used with objects, and is used to iterate over the properties of an object
