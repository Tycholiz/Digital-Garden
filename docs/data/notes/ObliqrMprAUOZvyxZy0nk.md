
###### Split vertically
`shift + |`

###### Split horizontally
`shift + _`

###### Close pane
`x`

###### Last active pane
`;`

###### Move current pane left/right
`{`/`}`

###### Pane zoom (cmd+shift+return in iTerm2)
`z`

###### Swap/rotate pane locations
`<`/`>`

###### Toggle between horitontal and vertical split
`<space>`

###### Break current pane out of window
`!`

###### Resize pane
`H`/`J`/`K`/`L`
