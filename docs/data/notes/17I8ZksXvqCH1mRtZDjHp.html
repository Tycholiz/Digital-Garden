Rendering Pipeline

The steps taken to transform and render a 3D model to 2D (the screen). There are APIs that can be used to control the underlying hardware and keep the programmer away from writing code to manipulate the graphics hardware accelerators .

  • some examples are Direct3D and OpenGL. The underlying hardware would be AMD/Intel/NVIDIA etc.

spec:(one of?) The most common type of rendering pipeline implementations is done with a shader.
