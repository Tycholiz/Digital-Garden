Cmds

Command line commands

note: all prefixed with prefix <C-a>, except where noted

  • ? see list of available commands

Sessions

  • d - detach
  • $ - rename session
  • (/) - swap between sessions
  • a - attach
    • -t 0 - attach to "session 0"
  • :new - new session
    • -s - add a name to the session
  • tmux new -s <session-name>
  • $ - rename session
  • s - list sessions
  • :kill-ses -t <session-name>
  • :attach -c /path/to/default/directory - change default directory of new windows

Windows

  • c - create new window
  • 1 - go to window 1
  • I - go to last window
  • <C-h>/<C-l> - swap between windows
  • & - close window

Panes

  • shift + | - split vertically
  • shift + _ - split horizontally
  • x - close pane
  • ; - last active pane
  • {/} - move current pane left/right
  • z - pane zoom (cmd+shift+return in iTerm2)
  • </> - swap/rotate pane locations
  • <space> - toggle between horitontal and vertical split
  • ! - break current pane out of window
  • H/J/K/L - resize pane

Plugins

  • I - fetch new plugins (like PlugInstall)
  • U - update plugins
  • opt+u - uninstall plugins not on list

Integration shortcuts

  • p - paste from clipboard
    • P - choose which buffer to paste from

Modes

Copy Mode (no prefix)

  • enter copy mode - prefix + [
  • move up/down a paragraph - {/}
  • go to middle - M
  • copy until end of line - D

Normal

  • paste from buffer - ]
  • send the contents of the current buffer to a temp file - P
    • custom bind
  • open current pane in vim - v
  • If you want to switch to a window based on something displayed in it (this also includes window names and titles but not history), (starting with more than one window open) press Ctrl-b f then type the string to search for and press Enter. You will be switched to a window containing that text if it's found. If more than one window matches, you'll see a list to select from.

Other

  • edit tmux.conf - e
